## 6p5Quest

**3 个月冲刺雅思 6.5 的数据驱动学习应用**：从单词与语法打底，覆盖听说读写练习、学习文档与课程视频；支持注册登录、学习进度记录，并基于学习行为实时估算雅思分数（Band Score）。

## 目标与定位

- **目标**：帮助用户在 12 周内系统提升至 IELTS 6.5（可配置个人目标分）
- **定位**：学习路径 + 练习系统 + 进度数据化 + 实时估分的一体化应用

## 核心功能（规划）

- **账号系统**：注册 / 登录 / 退出（后续可接入 OAuth）
- **学习路径**：单词 → 语法 → 听力 → 口语 → 阅读 → 写作（可自定义周计划）
- **练习系统**
  - 单词记忆与测验（复习间隔、错题本）
  - 语法点讲解与题库练习
  - 听力练习（音频、逐句、跟读）
  - 口语练习（录音、回放、打分维度）
  - 阅读练习（计时、定位题型）
  - 写作练习（作文管理、批改记录）
- **内容中心**：学习文档、课程视频（收藏、学习状态）
- **进度与数据**
  - 学习事件记录（练习、正确率、时长、间隔、完成度）
  - 看板（周/月趋势、薄弱项）
- **实时估分（特色）**
  - 根据用户练习记录与历史趋势，实时估算 Listening/Reading/Speaking/Writing 与总分
  - 提供“提升建议”：下一步最有效的任务与时间分配

> 说明：估分为统计/模型推断结果，用于学习决策辅助，不等同于官方考试结果。

## 技术栈

- **Web**：Next.js
- **DB / ORM**：Prisma + SQLite（本地开发）
- **UI**：Tailwind CSS
- **Testing**：Vitest
- **Tooling**：pnpm、ESLint
- **Container**：Docker

## 快速开始（本地开发）

### 前置要求

- Node.js 20+（建议使用 LTS）
- pnpm 9+

pnpm install
